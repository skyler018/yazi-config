# yazi-config

My personal [Yazi](https://github.com/sxyazi/yazi) configuration files.

## Contents
- `yazi.toml`: General configuration settings
- `keymap.toml`: Custom key bindings
- `theme.toml`: Appearance and theme configuration
- `package.toml`: Plugin and package management
- `flavors/`: Custom color flavors
- `plugins/`: Custom or community plugins
